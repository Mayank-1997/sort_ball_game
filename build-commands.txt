# Build Commands for Ball Sort Puzzle Android App

## Development Build
cordova build android

## Debug APK (for testing)
cordova build android --debug

## Release APK (for distribution)
cordova build android --release

## Run on connected device/emulator
cordova run android

## Run with live reload for debugging
cordova run android --livereload

## Generate signed APK for Play Store
# First, create keystore:
keytool -genkey -v -keystore ball-sort-release-key.keystore -alias ball-sort -keyalg RSA -keysize 2048 -validity 10000

# Then build release:
cordova build android --release -- --keystore=ball-sort-release-key.keystore --alias=ball-sort --storePassword=YOUR_PASSWORD --password=YOUR_PASSWORD
