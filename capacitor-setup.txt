# Alternative: Using Ionic Capacitor

## Install Capacitor
npm install -g @capacitor/cli
npm install @capacitor/core @capacitor/android

## Initialize Capacitor
npx cap init "Ball Sort Puzzle" "com.yourcompany.ballsort"

## Add Android platform
npx cap add android

## Copy web assets
npx cap copy

## Open in Android Studio
npx cap open android

## Sync after changes
npx cap sync
